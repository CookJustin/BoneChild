# Audio Assets

This directory contains the audio files for BoneChild.

## Required Files

### Background Music
- **7th realm.mp3** - Main background music (looping)
  - Place your MP3 file named "7th realm.mp3" in this directory

### Sound Effects (Optional)
The following sound effects can be added for enhanced gameplay:

- **attack.wav** or **attack.ogg** - Player attack sound
- **hit.wav** or **hit.ogg** - Enemy hit/damage sound
- **levelup.wav** or **levelup.ogg** - Level up sound effect
- **death.wav** or **death.ogg** - Enemy death sound

## Supported Formats
- Background Music: MP3, OGG
- Sound Effects: WAV, OGG

## Notes
- Background music will loop automatically
- Sound effects are optional - the game will run without them
- Volume is set to 50% for background music and 60% for sound effects by default
